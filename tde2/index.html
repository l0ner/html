Trinity Desktop Environment Version 3.5.13 Released!

2011-11-01 by Timothy Pearson

Trinity Desktop Environment version 3.5.13 has been released! This version provides significant enhancements to areas such as desktop security, GTK application support, and general usability.

You will find a detailed list of upgrades and improvements to the Trinity system here: http://www.trinitydesktop.org/wiki/bin/view/Documentation/Releases_3_5_13
